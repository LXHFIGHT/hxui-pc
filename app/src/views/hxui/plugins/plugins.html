Plugins

HXUI 全局对象

在引入 hxui.min.js 文件后，即可使用包括表单验证器，模态框，日历选择器等插件，调用方式可以直接使用 HXUI.functionName 直接调用。

# smartValidate

HXUI.smartValidate 是一个易用的表单验证方法，可用于对一个选择的父组件（默认为body）中所有带有特定属性的子组件进行判断并将结果返回。

# confirm

使用 HXUI.confirm 方法弹出一个带有确定和取消的确认框。 
   HXUI.confirm ({
       title: '确定删除吗',    // 确认信息
       confirm: function() {
            // 确认按钮触发事件
       },
       cancel:  function() {
            // 取消按钮触发事件【可选】
       }
   })
